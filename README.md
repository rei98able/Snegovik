# Kinotech App

Социальная сеть, где пользователи могут составлять свои списки любимых фильмов, обмениваться этими списками с другими пользователями, а также получать рекомендации.

- [Парамонова Анна](https://github.com/ananasann)

## Стек
### 1. Архитектура: MVVM
### 2. Базы данных: Firebase Realtime Database
### 3. Запросы в сеть: Retrofit + Coroutines
### 4. Многопоточность: Coroutines
### 5. Фотографии: Glide
### 6. А также: Data Binding, Material Design, Toast, etc.
